جهت راست چین کردن نوشته ها ابتدا کلیدهای سمت راست‌ کیبورد را بزنید : SHIFT+CTRL


دستورات را کپی کرده و بجای paste کردن در محیط برنامه فقط راست کلیک کنید تا انتقال یابد.

ابتدا نرم افزار putty  را اجرا کنید، سپس IP سرور را وارد کرده تا وارد محیط برنامه شوید.

دستور اول :
اگر یوزرنیم سرور چیزی غیر از root بود ابتدا این دستور را وارد کنید، در غیر اینصورت نیازی به این دستور نیست :
sudo -i

اگر یوزرنیم شما root است از اینجا شروع کنید:
apt update
--------------------------------------------------------------------------------------------
دستور دوم : 
wget -N --no-check-certificate https://raw.githubusercontent.com/sfc9982/AnyConnect-Server/main/ocserv-en.sh
chmod +x ocserv-en.sh
bash ocserv-en.sh
--------------------------------------------------------------------------------------------
دستور سوم : 
wget -N --no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh && chmod +x bbr.sh && bash bbr.sh

--------------------------------------------------------------------------------------------
افزودن کاربر :
جهت مدیریت کاربران ( افزودن - حذف - غیرفعال کردن کاربر)، ریستارت  کردن و ... ابتدا میبایست این دستور را وارد کنید. 
bash ocserv-en.sh

--------------------------------------------------------------------------------------------
تغییر پورت : 
nano /etc/ocserv/ocserv.conf
